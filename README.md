## Запуск

1. Клонируйте репозиторий:

```bash
git clone https://github.com/roadtoseniors/PR6.3.git
```

2. Откройте решение (`.sln`) в Visual Studio.

3. Соберите проект: **Build → Build Solution** (или `Ctrl+Shift+B`).

4. Запустите приложение: **Debug → Start** (или `F5`).

---

## Запуск тестов

Тесты находятся в проекте `UnitTestProject`. Для их запуска:

- В Visual Studio: откройте **Test Explorer** (`Test → Test Explorer`) и нажмите **Run All**.
- Через CLI:

```bash
dotnet test
```
